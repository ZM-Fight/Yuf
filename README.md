#### Yuf项目，2014年9月到11月
##### Yuf是一款连通连锁超市和社区白领，为白领提供定制食材配送服务的手机应用。
##### 涉及volley网络框架、json传输数据、ShareSDk组件。用到了瀑布流、PullToRefresh 等开源工具。
